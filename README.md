# DevTrack

DevTrack é um sistema web para **gerenciamento de projetos e tarefas**, desenvolvido como projeto de portfólio.  
Este repositório contém o **frontend da aplicação**, responsável pela interface do usuário e pela integração com uma API REST autenticada com JWT.

## Stack
React, JavaScript, React Router DOM, Axios, CSS

## Principais Funcionalidades
- Autenticação de usuários (login e cadastro)
- Gerenciamento de projetos (criar, listar, editar e excluir)
- Gerenciamento de tarefas por projeto
- Atualização de status das tarefas (pendente, em andamento e concluída)
- Rotas protegidas e comunicação segura com o backend

O frontend consome a **DevTrack API**, utilizando autenticação JWT e seguindo boas práticas de organização e usabilidade.
